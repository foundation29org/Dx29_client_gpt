🔍 ANÁLISIS DE ARQUITECTURA Y PATRONES
═══════════════════════════════════════════════════════════════════════════════════════════

🏗️ PATRONES DE DISEÑO IDENTIFICADOS:
1. **BEM Methodology**: Implícito en nombres como .footer-column, .footer-title, .footer-links
2. **Atomic Design**: Elementos → Componentes → Organismos
3. **Mobile First**: Media queries progresivos (576px → 768px → 992px)
4. **Component-Scoped Theming**: Variables SCSS para consistencia
5. **State-Driven Styling**: Estados hover, focus, disabled bien definidos

🎨 SISTEMA DE COLORES:
- **Primarios**: $dark-red (#8B0000), $medium-red (#B22222)
- **Neutrales**: $dark-gray (#2C2C2C), grises varios (#9CA3AF, #E5E7EB)
- **Fondos**: $footer-dark (#1a1d23), $footer-darker (#151820)
- **Funcionales**: #DC2626 (error), #40c057 (success - referenciado globalmente)

🔧 TÉCNICAS AVANZADAS SCSS:
- **Nesting**: Hasta 4 niveles de profundidad
- **Parent Selectors**: Uso extensivo de &:hover, &:after, &:before
- **Pseudo-elementos**: Para decoraciones (:after) y overlays (:before)
- **Interpolación**: Variables integradas en valores complejos
- **Media Queries**: Anidados dentro de selectores

⚠️ PROBLEMAS TÉCNICOS DETECTADOS:
1. **::ng-deep Deprecado**: Usado extensivamente, necesitará migración
2. **Especificidad Alta**: Muchos !important, podría causar problemas futuros
3. **z-index Management**: No hay sistema centralizado de z-index
4. **Duplicación**: Patrones repetidos que podrían ser mixins
5. **Magic Numbers**: Valores hardcoded sin explicación (56px, 484px)

🔗 DEPENDENCIAS EXTERNAS:
- **Angular Material**: Checkboxes, Form Controls
- **Bootstrap**: Grid system, modal structure
- **FontAwesome**: Iconos (fa-, fab-)
- **NgBootstrap**: Modal system
