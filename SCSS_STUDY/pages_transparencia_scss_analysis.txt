```scss
Variables SCSS:
- $dark-red: #8B0000       - Color brand principal

- $dark-gray: #2C2C2C      - Texto principal
- $light-gray: #F8F9FA     - Fondos suaves
- $white: #FFFFFF          - Contrastes

Colores adicionales:
- #1a1a1a                 - Final degradado hero
- #E0E0E0                 - Bordes cards
- #444444                 - Texto párrafos
- #666666                 - Texto secundario
- rgba(139, 0, 0, 0.2)    - Bordes translúcidos
```

### 📐 SISTEMA DE ESPACIADO
```scss
Padding sections:
- 6rem 0 4rem (96px 0 64px)  - Hero desktop
- 4rem 0 (64px)              - Main content
- 2rem (32px)                - Cards padding

Margins:
- 4rem (64px)                - Entre content sections
- 2rem (32px)                - Estándar elementos
- 1.5rem (24px)              - Headers y elementos menores

Heights fijas:
- 120px → 100px → 80px       - Sponsor cards responsive
```

### 📱 SISTEMA RESPONSIVE
```scss
Breakpoints:
- 768px                      - Tablets (Bootstrap md)
- 576px                      - Móvil pequeño (Bootstrap sm)

Font-size progression:
Hero title: 3rem → 2.2rem → 1.8rem
H2: 2rem → 1.5rem
H3 collaboration: 1.4rem → 1.2rem → 1.1rem
Párrafos: 1.05rem → 1rem → 0.95rem
```

### 🎨 FILTROS Y EFECTOS
```scss
Grayscale strategy:
- Partners: 20% grayscale (mantiene 80% color)
- Sponsors: 30% grayscale (mantiene 70% color)
- Hover: 0% grayscale (color completo)

Transform effects:
- Logo scale: 1.05 (partners) vs 1.1 (sponsors)
- Card elevation: translateY(-5px) estándar
```
